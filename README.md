
## How to Play

1. Select your difficulty level (Easy/Medium/Hard)
2. Read the movie description
3. Use hints if needed (limited based on difficulty)
4. Type your guess and submit
5. Score points and move to the next movie!

## Difficulty Levels

| Level  | Hints | Timer |
|--------|-------|-------|
| Easy   | 5     | 120s (optional) |
| Medium | 3     | 60s (optional) |
| Hard   | 1     | 30s (always on) |
